# 组件的使用

  1. 一般组件需要定义一个文件夹，文件夹命名为 components

  2. 文件夹下可以定义一个需要引用的组件

  3. 文件夹下包含 4 个文件分别是 js、json、wxml、wxss。

  4. 控制组件的显示与隐藏通过 hidden 来控制，true 为隐藏，false 为显示

  5. 注意：需要在 json 文件中定义两个属性：

  • 组件在小程序中的使用方法如下：
  • 在需要引用的页面 json 文件中写入:
"usingComponents": {
    // loadmore自定义组件名
"loadmore": "/components/loadmore/loadmore"

}

# 组件上的自定义属性

  • 一般用来做详情页

在组件里定义一个属性 data-xx，xx 即为自定义的属性名，加上要传入的值。

注意: data-list-name命名的时候在获取的的时候要使用 listName

// index.wxml
<view class="td">
  <text bindtap="update" data-name="{{item.userName}}"></text>
</view>

# 父传子（类似自定义组件上的自定义属性）

  1. components 中创建两个组件 b a 分别当作父子组件

  2. 在父组件 b 中 json 文件引入并声明

// b.json
  "component": true,
  "usingComponents": {
    "aaa":"/components/aaa/aaa"
  }
