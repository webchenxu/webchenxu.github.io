# let 命令

  • ES6 新增了 let 命令，用来声明变量。它的用法类似于 var，但是所声明的变量，只在 let 命令所在的代码块内有效。
{
  let a = 10;
  var b = 1;
}

a; // 未定义
b; // 1
// let声明的变量只在它所在的代码块有效。

for 循环的计数器，就很合适使用 let 命令。

for (let i = 0; i < 10; i++) {
  // ...
}

console.log(i); // i is not defined

上面代码中，计数器 i 只在 for 循环体内有效，在循环体外引用就会报错。

下面的代码如果使用 var，最后输出的是 10。

var a = [];
for (var i = 0; i < 10; i++) {
  a[i] = function() {
    console.log(i);
  };
}
a[6](); // 10
