# 扩展运算符

含义: 扩展运算符（spread）是三个点（...）。它好比 rest 参数的逆运算，将一个数组转为用逗号分隔的参数序列。
用途：
1、将一个数组变为参数列表
2、数组拼接
3、使用 push 将一个数组添加到另一个数组的尾部
4、可以将伪数组转换成真数组

# forEach()

forEach 会遍历数组，没有返回值,不允许在循环体内写 return, 不会改变原来数组的内容

let arr = [1, 2, 3, 4, 5, 6, 7];
//forEach方法会改变原数组
arr.forEach((item, index, arr) => {
  arr[index]++;
});
console.log(arr); //[2, 3, 4, 5, 6, 7, 8]

# map()

map 遍历数组，会返回一个新数组，不会改变原来数组里的内容

let arr = [1, 2, 3, 4, 5, 6, 7];
let arr1 = arr.map((item, index, arr) => {
  return --arr[index];
});
console.log(arr1); //[0, 1, 2, 3, 4, 5, 6]

# reduce()

reduce: 这个方法接受一个函数作为一个参数，这个函数也称作函数累加器，数组的每一项从左往右一次减少直到最后一次运算，最终返回一个结果 reduce((prev, next, index, arr)=>{},prev 初始值） prev: 参与运算得初始值，或者上一次运算得返回结果 next：依次参与运算的数组得每一项 index: 索引 arr: 原数组 用途：给数组求和

const arr = [1, 2, 3, 4];
let Arr = arr.reduce((prev, next, index, arr) => {
  //做运算
  console.log(prev); // 1      undefined
  console.log(next); // 2      3
  console.log(index); // 1 索引 2
  console.log(arr); // arr    arr
  return prev + next; // 1+2+3+4
  //初始：  prev： 1  next: 2  1+2=3
  //第二次：prev： 3  next：3  3+3=6
  //第三次  prev： 6  next：4  6+4=10
  //第四次  prev： 10 next  无  此次不计算
});
console.log(Arr);

# every()

every 遍历数组，每一项都是 true, 则返回 true, 只要有一个是 false, 就返回 false

let array = [1, 2, 3, 4];
let a = array.every((item, index, array) => {
  return item > 2;
});
console.log(a); // false;
